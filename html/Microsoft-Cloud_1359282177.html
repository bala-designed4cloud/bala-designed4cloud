Microsoft has evolved constantly over several decades from just an Operating System and Microsoft Office software supplier to providing fully Cloud based identity and Collaboration services globally over several regions to anyone as far as you have internet connectivity.

Identify and Access Management - Azure AD

Traditionally if all staff are in office locations, we used to protect valuable IT assets using network controls, but with anyone from anywhere, the identity of the user and how to make sure the identity is always protected, validated and re-validated is critical to enable access to valuable IT assets.

Therefore, as a first step to create a virtual office, it’s critical to define and setup identities and permissions to the users before we proceed with anything else.

The services provided as part of identity and access management includes:

  • Authentication - Username/Password with MFA

  • Authorisation - Azure AD Group Membership managed

  • Device Management - Protect devices as another identity so it can be protected against known vulnerabilities - Optional

  • Hybrid Identity - If you already Microsoft Active Directory used with your office location and if you wanted to extend to virtual office, while keeping the office location - Optional

  • Advance identity protection including Machine Learning and Privilege Access Management - Optional.

Web based Business Applications - Microsoft 365

The use of multiple devices such as desktop, laptop, tablet and mobile has influenced the use of web browser to access all business applications such as outlook, office and sharepoint and that trends continue with virtual office with bring you own device as well.

The services provided as part of this includes:

  • Email collaboration with the mailbox with your chosen custom domain name for email addresses.

  • To share your files and synchronise your folders to cloud, OneDrive for each user.

  • To author and publish high quality documents, it provides all Office 365 applications such as Word, Powerpoint, Excel and others as part of the Microsoft Office family.

  • To interact with teams via chat, voice and video and at the same time share content, it also provides Microsoft teams and Microsoft sharepoint.

As this is the web based offering, all of the services are accessible via web browser and only via web browser. You will need good internet connectivity to use these services.

Local Applications - Office 365
